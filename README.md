# Briskit 🎟️

Briskit is a portfolio project showcasing an event management and ticketing system.  
Users can browse events by city, while admins have exclusive access to add and manage events.

---

## ✨ Features
- Public view for browsing events by city
- Admin dashboard with login
- Add new events with category, description, poster, and ticket price
- SQL backend (Supabase/Postgres or PlanetScale/MySQL)
- Responsive frontend built with HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: SQL (Supabase/Postgres or PlanetScale/MySQL)
- Version Control: Git + GitHub
